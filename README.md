# About Me

I design and deliver robust embedded software for Automotive and IoT systems. My expertise includes Embedded C, AUTOSAR, RTOS, device communication protocols, and Linux device driver work.

I also explore the intersection of AI and embedded systems through automation, tooling, and workflow improvements that make engineering teams more effective.

### What I do

- Develop production-grade embedded firmware and automotive software
- Troubleshoot low-level issues across hardware/software boundaries
- Build secure device communication using CAN, LIN, UART, SPI, and I2C
- Apply AI/automation techniques to improve engineering productivity
